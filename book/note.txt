引用Vue要加上 : 
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

使用bootstap的功能，要加上 :
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.0/css/bootstrap.min.css">
例如 : <button type="button" class="btn btn-success"> 或 btn btn-danger
例如2 : <div class="container"><div class="row">...

以下是Font Awesome的用法 : 
加上 :
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css">
就可使用
<i class="fa fa-plus"></i>    做出加號的效果
<i class="fa fa-minus"></i>   做出減號的效果
做出笑臉 :
<script src='https://kit.fontawesome.com/a076d05399.js'></script>
<i class="far fa-smile-beam"></i>
網址 : https://www.w3schools.com/icons/fontawesome5_icons_emoji.asp
